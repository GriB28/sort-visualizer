### ℹ️ Описание функционала

<img src="/.readme/3.webp" width=80 align="left" style="float: left; margin-right: 8px;" alt="Main menu commands">

<br>

_Консольное приложение для визуализации работы алгоритмов сортировки._

<br>

💠 Программа представляет собой консольное приложение с интерактивным меню.

Она позволяет:
- автоматически компилировать бинарный файл сортировщика через CMake и исправлять ошибки в случае их возникновения;
- генерировать сортировки с пользовательской конфигурацией;
- запускать выбранные сортировки массива;
- рендерить видео с визуализацией процесса сортировки;
- автоматически создавать виртуальное окружение и управлять необходимыми зависимостями;

Основные команды приложения:
- `main` — полный цикл: генерация массива, сортировка, рендер видео;
- `generate` — генерация конфигурации сортировки;
- `sort` — запуск алгоритма сортировки;
- `render` — рендер визуализации в видео;
- `reset` — очистка временных файлов и сброс настроек;
- `help` — справка по командам;
- `exit` — выход из программы.


#
### 🚀 Инструкция по запуску программы

<img src="/.readme/4.webp" width=80 align="left" style="float: left; margin-right: 8px;" alt="Main menu commands">
<br>

_Как запустить и программу и работать с ней?_
<br>
<br>


1. Запустить основной скрипт приложения [`run.sh`](./run.sh)

2. После запуска откроется интерактивное меню команд.

3. Введите одну из доступных команд:

   - `main` — выполнить полный цикл работы программы (генерация массива, сортировка и рендер видео);
   - `generate` — создать конфигурационный файл для сортировки;
   - `sort` — выполнить сортировку по заданным параметрам;
   - `render` — создать видео с визуализацией сортировки;
   - `reset` — сбросить настройки CMake и VENV и очистить временные файлы в случае возникновения ошибок;
   - `help` — открыть справку по работе с приложением;
   - `exit` — закрыть приложение.

4. Следуйте инструкциям, которые программа выводит в терминале.

5. Сгенерированные видеоролики будут сохранены в папке `videos` в корне проекта.

✅ Готово! Программа готова к использованию.

### ☑️ Установка и конфигурация
<img src="/.readme/1.webp" width=80 align="left" style="float: left; margin-right: 8px;" alt="Launch and configuration">

_Инструкции по установке и настройке._

Приложение адаптировано под ОС Windows и Linux.

- Windows 10+ (WSL) и Linux:

  * (!) Для запуска программы на ОС Windows необходимо заранее установить Windows Subsystem for Linux (WSL). Как это сделать: [инструкция](https://learn.microsoft.com/ru-ru/windows/wsl/install);

  * Должны быть установлены необходимые пакеты: `g++`(если не установлен), `python3`(если не установлен), `cmake`, `dos2unix`, `python3-venv`

  * Перед запуском сборщика [`run.sh`](./run.sh) необходимо конвертировать его в формат Unix: `dos2unix run.sh`

  * Запуск приложения: `./run.sh`
